# Github

  1. vue-awesome-swiper (opens new window):一个功能丰富的轮播组件

  2. surmon.me (opens new window):使用vue开发的一个风格独特的博客。博客地址：blog (opens new window)

  3. HelloGitHub (opens new window) 分享 GitHub 上有趣、入门级的开源项目。月刊：link

  4. best-resume-ever (opens new window) has multiple beautiful resumes,Made with Vue and LESS. example (opens new window)

  5. scrollama (opens new window) is a modern & lightweight JavaScript library for scrollytelling using IntersectionObserver (opens new window) in favor of scroll events. example (opens new window)

  6. librephotos (opens new window)一个自托管的相册应用，后端是 Django，前端是 React。example (opens new window)

  7. ThibaultJanBeyer:dragSelect (opens new window) 简单的javascript拖动选择功能插件，类似 window 系统文件拖动选择效果，不依赖任何第三方项目

  8. Underscore (opens new window)一个 JavaScript 实用库，提供了一整套函数式编程的实用功能，但是没有扩展任何 JavaScript 内置对象。文档 (opens new window)

  9. Semantic (opens new window) 是一个能够帮助你创建漂亮的响应式HTML的人性化开发框架。 Github (opens new window) 最后更新时间：on 22 Oct 2018
